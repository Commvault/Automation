VirtualServer.SNAPUtils.vsa_snap_templates module¶

Template file for performing IntelliSnap operations for Virtual Server Agent

VSASNAPTemplates is the only class defined in this file

VSASNAPTemplates: Template class to perform IntelliSnap operations

VSASNAPTemplates:

__init__() – initializes VSASNAPTemplates object

vsasnap_template_v1() – Template to Perform Snap backup, backup copy and Restore

operations for VSA V1 client.

Steps: (if run_all_operation input is not provided)

  1. Snap Backup of VM coming from a particular snap vendor array

  2. Backup copy of that VM.

  3. Guest file restore from Snap (live Browse).

  4. Out of place full vm restore from snap.

  5. Guest file restore from backup copy.

  6. In place full vm restore from backup copy

Steps: (if run_all_operation input is provided)

  1. Snap Backup of VM coming from a particular snap vendor array

  2. Backup copy of that VM.

  3. Guest file restore from Snap (live Browse).

  4. Disk restore from Snap.

  5. Out of place full vm restore from snap.

  6. In place full vm restore from snap

  7. Guest file restore from backup copy.

  8. Disk restore from backup copy

  9. Out of place full vm restore from backup copy

  10. In place full vm restore from backup copy

vsasnap_template_v2() – Template to Perform Snap backup, backup copy and Restore

operations for VSA V2 client.

Steps: (if run_all_operation input is not provided)

  1. Snap Backup of VM coming from a particular snap vendor array

  2. Backup copy of that VM.

  3. Out of place full vm restore from snap (parent job)

  4. Guest Files restores - v2 Indexing from snap

  5. FULL VM out of Place restores : v2 Indexing from snap

  6. Guest Files restores - v2 Indexing from backup copy

  7. FULL VM out of Place restores - v2 Indexing from backup copy

Steps: (if run_all_operation input is provided)

  1. Snap Backup of VM coming from a particular snap vendor array

  2. Backup copy of that VM.

  3. Guest file restore from Snap (live Browse).

  4. Disk restore from Snap. -> Can be added

  5. Out of place full vm restore from snap.

  6. In place full vm restore from snap -> Can be added

  7. Guest file restore from backup copy.

  8. Disk restore from backup copy -> Can be added

  9. FULL VM out of Place restores from backup copy

  10. In place full vm restore from backup copy -> Can be added

  11. Guest Files restores - v2 Indexing from snap

  12. FULL VM out of Place restores : v2 Indexing from snap

  13. Guest Files restores - v2 Indexing from backup copy

  14. FULL VM out of Place restores - v2 Indexing from backup copy
