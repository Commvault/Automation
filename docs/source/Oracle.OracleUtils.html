db_shutdown(mode)[source]¶

Shut down the database

Args:
mode (str) – Mode of shutdown to be given to the database

SHUT_TRANSACTION : shutdown using TRANSACTIONAL SHUT_FINAL : shutdown using FINAL SHUT_ABORT : shutdown using ABORT SHUT_IMMEDIATE : shutdown using IMMEDIATE

Raises:
ValueError:

– If the database connection wasn’t established – If the mode for shutdown is invalid

DatabaseError:

– Exception in shutting the database down
