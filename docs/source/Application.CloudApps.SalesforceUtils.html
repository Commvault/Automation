Initiates restore to database job and waits for completion Args:

restore_options (dict) – provide restore options like destination info,

list of objects to restore Example:

restore_options = {

‘object_list’: /Objects/Account, ‘dest_client_name’: destination client name

}

salesforce_restore_options (dict) – provide salesforce specific options
Example:
salesforce_restore_options = {

‘destination_path’: staging path location, ‘db_type’: SQLSERVER or POSTGRESQL, ‘db_host_name’: database hostname, ‘db_instance’: database instance name, ‘db_name’: database name, ‘db_port’: port of the database, ‘db_user_name’: database user name, ‘db_user_password’: database user password, ‘dependent_level’: restore dependent children

level,

‘streams’: number of streams, ‘copy_precedence’: copy from which restore needs

to be done,

‘from_time’: start time for browse range, ‘to_time’: end time for browse range

}

Returns:
