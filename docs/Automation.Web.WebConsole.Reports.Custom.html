class DatabaseDataset¶

Bases: Automation.Web.WebConsole.Reports.Custom._datasets._DatabaseDatasetQueryTab, Automation.Web.WebConsole.Reports.Custom._datasets._DatabaseDataSource, Automation.Web.WebConsole.Reports.Custom._datasets._ParametersTab, Automation.Web.WebConsole.Reports.Custom._datasets._AdvancedTab, Automation.Web.WebConsole.Reports.Custom._datasets._FieldsTab

Mixin class used for database type Dataset, all operations on the database type class has to be done using this class

Examples:

db_dataset = DatabaseDataset(webconsole_object)

builder_object.add(db_dataset)
# above line takes care of opening and closing the Dataset.
# Dataset object is not usable without adding it to the
# Builder object, and the methods will work only when the
# Dataset modal is opened

# Below method will be provided by _DatabaseDataSet implementation
db_dataset.set_dataset_name("HardRock Testing")

# Below method will be provided by _dataset.DataSource implementation
db_dataset.set_all_commcell_datasource()
