Main file for Server module type(s) handlers

Server.serverhelper module¶

Main file for performing Server related operations on Commcell

ServerTestCases: Provides test case related operations.

ServerBase: Provides common commserve related operations.

ServerTestCases:

__init__() – initialize instance of the ServerTestCases class

__repr__() – Representation string for the instance of the
ServerTestCases class for specific test case

fail() – Marks the test case failed and logs the errors

ServerBase:

__init__() – initialize instance of ServerBase class

__repr__() – Representation string for the instance of the
ServerBase class for specific test case

check_client_readiness() – performs check readiness on the list of clients

restart_services() – restarts services on the list of clients
