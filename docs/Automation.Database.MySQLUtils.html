Helper file for performing MYSQL operations

MYSQLHelper is the only class defined in this file

MYSQLHelper: Helper class to perform MYSQL operations

MYSQLHelper:

__init__() – initializes MYSQLHelper object

run_backup() – initiates backup job for specified subclient and
waits for completion

get_mysql_db_password() – gets the MySQL Server Password

get_databases_size() – gets the size of the databases

get_database_and_table_sizes() – calls get_databases_size() and get_tables_list_for_db()
and returns
compare_db_and_table_sizes() – compare the database and table sizes after backup
and restore

drop_table() – drops the table from MySQL Server

clean_up_tables() – takes a dictionary as input and calls drop_table()

basic_setup_on_mysql_server() – checks for basic setups on MySQL Server

get_tables_list_for_db() – gets the existig table list for a database

get_table_size() – gets the table size

log_bin_on_mysql_server() – checks whether binary logging is enabled in MySQL Server
or not

create_data() – creates the data in tables created by automation

create_table_with_text_db() – create tables in a database
